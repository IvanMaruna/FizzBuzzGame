# FizzBuzzGame

Denne program implementerer FizzBuzz matematikkleken.

## Beskrivelse

Konsoll-vindu program delt inn i tre deler.
* Del 1 er på bransje master. Det er et program som implementerer matematikkleken FizzBuzz
* Del 2 er på bransje Del2. Programmet ble utvidet så at det kan kjøres fra 100 til 1 med nye regler som erstatter Fizz og Buzz. "Jazz" for tall som er delelig med 9, og 
"Fuzz" for tall som er delelig med 4.
* Del 3 er på bransje Del3. Programmet ble utvidet så at reglene for spillet hentes fra en ekstern tjeneste. Vi har to tjenester Statisk og Dynamisk. Programmet kan kjøres for begge tjenestene hvis du legger til et argument når du kjører det i terminalen.

### HOW-TO kjøre programm

For statisk tjeneste:
```
dotnet run
```
For dynamisk tjeneste:
```
dotnet run dynamic
```
